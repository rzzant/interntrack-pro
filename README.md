# InternTrack

## Live Demo

https://interntrack-pro-ten.vercel.app/

## Overview

InternTrack is a full-stack web application designed to help students efficiently track and manage their internship applications with analytics and insights.

## Problem

Students apply to multiple internships but struggle to track progress, deadlines, and outcomes.

## Solution

InternTrack provides a centralized dashboard to:

* Track applications (Applied → Interview → Offer)
* View analytics of success rate
* Manage deadlines and follow-ups

## Features

* Application tracking dashboard
* Analytics and insights
* Deadline reminders
* Authentication system
* Organized application history
